## Цель проекта:

Разработать систему персонализированных рекомендаций постов для пользователей социальной сети с учетом их поведения и характеристик контента

## Задачи проекта:

Выгрузить данные из баз данных и провести анализ полученных данных

Сформировать новые признаки, которые могут быть полезны для модели

Обучить модель ранжирования постов

Оптимизировать качество рекомендаций по метрике HitRate@5

Реализовать сервис для выдачи рекомендаций в реальном времени

## Данные:

Базы данных Карпов Курсы

## Результаты исследования:

Обучена модель рекомендаций с метрикой HitRate@5 > 0.52

Реализован сервис, выдающий топ-5 постов для пользователя

Время ответа сервиса < 0.5 секунды

Оптимизировано использование памяти (< 4 ГБ)
